Oxidizer

Last Autumn, I took a digital sound and music course which culminated in this project: a program for applying various types of distortion to an arbitary sample of audio.
I created this as an actual tool to augment my musical process. In my spare time, after computer science, creating and mixing music is my passion. I was unsatisfied with the distortion capabilities of the existing digital music tools I'm very familiar with, so I used a few well-regarded Python libraries to make this app. It allows the user apply any standard audio distortion you could think of as well as some unique digital distortion techniques that aren't available in any commercial software. I also used a Python TUI library for the first time in this project, so the final result was a unique audio tool that's as powerful as it is accessible.

Music
